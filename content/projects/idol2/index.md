## 프로젝트 개요
웹 기반 MMD 모션 에디터. 프레임별로 VMD 모션을 조정하고 편집하는 도구.

## 핵심 기능 (계획)
- VMD 로드 및 타임라인 표시
- 프레임별 본 위치/회전 편집
- 3D 뷰포트에서 실시간 미리보기
- 키프레임 추가/삭제/이동
- 보간 곡선 편집
- 편집된 VMD 내보내기
- PMX 모델 로드하여 모션 미리보기

## 기술 스택 (예정)
- Next.js + Tailwind CSS (Lab0 내부 페이지)
- Three.js (3D 뷰포트)
- mmd-parser (VMD 파싱)
- 자체 VMD 직렬화
