## 프로젝트 개요
웹 기반 PMX 에디터. 머티리얼, 텍스처, 모프 등을 편집하고 저장하는 도구.

## 핵심 기능 (계획)
- PMX 로드 및 3D 뷰
- 파츠별 표시/숨기기
- 머티리얼 색상 편집 (실시간)
- 텍스처 교체 (이미지 업로드)
- 모프 슬라이더 (표정, 체형)
- 투명도 조정
- 수정된 PMX 다운로드 (내보내기)

## 확장 기능 (추후)
- 본 위치 편집
- 의상 PMX 머지
- 물리 설정 편집

## 기술 스택 (예정)
- Next.js + Tailwind CSS (Lab0 내부 페이지)
- Three.js (3D 뷰포트)
- mmd-parser (PMX 파싱)
- 자체 PMX 직렬화
