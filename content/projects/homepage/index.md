## 개요

## 기술 스택
- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Vercel 배포
- Markdown 기반 콘텐츠 관리

## 설계 방향
- 심플하고 아담한 화이트톤 UI
- 모바일 퍼스트 반응형
- Git이 곧 CMS (Markdown 파일 push → 자동 반영)
- 초기에는 DB 없이 시작, 필요 시 Turso 연동
